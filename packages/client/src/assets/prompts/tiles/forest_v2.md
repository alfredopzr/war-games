# Terrain Tile — Forest (Variant 2)

## Subject
A dense cluster of dark trees viewed from isometric perspective. This variant has a narrow dirt path cutting through the trees — an old access road now barely visible, overgrown at the edges. The trees lean slightly, stressed and partially burned. One large tree has fallen across the path, trunk snapped partway up. Dead leaves carpet the ground between tree roots.

## Human Presence
Forgotten traces. The access road is being erased by vegetation. The fallen tree blocks what was once a clear route. Infrastructure is losing to entropy.

## Hex Shape
Flat-top hexagon tile, isometric perspective. The tree canopy extends slightly above the hex boundary. The base ground and path fit within the hex footprint.

## Color Palette — Ash & Ember
- Canopy: charred green (#3A4030), some branches leafless
- Fallen tree: dark grey-brown, exposed pale wood at the break point
- Path: dark packed earth, slightly lighter than surrounding ground
- Ground cover: dark leaf litter, dead brown
- Tree trunks: near-black

## Art Style
Isometric hex tile sprite. Visually distinct from forest_v1 — the path and fallen tree create a different composition. Still reads as "dense forest" at game zoom. Dark, moody palette. The path adds a subtle human trace element.

## World Context
Ash & Ember setting. This was a logging road or construction access route. After the collapse, nobody maintained it. Trees fell, vegetation encroached, and the road is disappearing season by season.

## Output
- Transparent PNG
- Isometric hex tile, approximately 120x140 pixels
- Flat-top hexagon orientation
- Tree canopy may extend slightly above the hex top edge for visual depth
- No units — terrain only
